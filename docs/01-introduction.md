# Module 1: Introduction to Cloudflare

**Duration:** 30 minutes

## What You'll Learn

- Cloudflare Platform basics
- Edge Computing and Serverless concepts
- DNS Security and Zero Trust overview

## Steps

### 1. Create Cloudflare Account

1. Go to https://dash.cloudflare.com
2. Click "Sign Up"
3. Enter your email address
4. Create a password
5. Verify your email
6. Log in to dashboard

### 2. Explore Dashboard

**Main Dashboard Sections:**

- **Home** - Overview and quick actions
- **Websites** - Domain management
- **Workers & Pages** - Deploy applications
- **Zero Trust** - Security features

### 3. Key Concepts

**Edge Computing:**
- Code runs on Cloudflare's global network (300+ locations)
- Faster response times
- No servers to manage

**Serverless:**
- Write code without managing infrastructure
- Pay only for what you use
- Automatic scaling

**DNS Security:**
- Filter malicious domains
- Block unwanted content
- Protect users from threats

**Zero Trust:**
- Secure access without VPN
- Verify every request
- Policy-based access control
