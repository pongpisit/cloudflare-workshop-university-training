### 4. Zero Trust Network Access (ZTNA)

#### English

**Old way: VPN**

Problem: If VPN is compromised, attacker gets full access!

**Key principles:**

**Simple explanation:**

- A master key to your entire building
- They can go anywhere
- If key is stolen, big problem!

**Zero Trust is like:**
- Giving specific room keys only
- They can only access what they need
- Each door checks their identity
- Much safer!

**Benefits for universities:**
- No VPN client installation needed
- Access from any device, anywhere
- Better security
- Easier to manage
