# Screenshots Directory

This directory contains all screenshots for the workshop modules.

## Structure

```
screenshots/
├── module1/     # Introduction screenshots
├── module2/     # Cloudflare Pages screenshots
└── module3/     # DNS Security screenshots
```

## Guidelines

- **Format:** PNG
- **Resolution:** 1920x1080 or 1440x900
- **File size:** < 500KB per image (use compression)
- **Naming:** Use descriptive names with numbers (e.g., `01-dashboard-home.png`)

## How to Add Screenshots

1. Take screenshot following [SCREENSHOT-GUIDE.md](../../SCREENSHOT-GUIDE.md)
2. Save to appropriate module folder
3. Compress if > 500KB
4. Commit and push to repository
