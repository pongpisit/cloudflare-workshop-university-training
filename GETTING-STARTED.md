### English

**Problem: Can't create account**
- Solution: Make sure you're using a valid email address
- Try a different browser
- Check if your email provider is blocking Cloudflare emails

**Problem: Didn't receive verification email**
- Solution: Check spam/junk folder
- Wait 5 minutes and try again
- Click "Resend verification email" on login page

- Solution: Cloudflare updates their interface regularly
- The core features remain the same
- Ask your instructor if you can't find something

**Problem: Browser is slow**
- Solution: Close other tabs and programs
- Clear browser cache
- Restart your browser
